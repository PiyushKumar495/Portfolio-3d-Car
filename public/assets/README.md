# Assets Directory

Place your images and logos here:

## Structure

```
assets/
├── logos/
│   ├── dotnet.svg
│   ├── csharp.svg
│   ├── azure.svg
│   ├── docker.svg
│   ├── react.svg
│   └── nextjs.svg
└── projects/
    ├── ecommerce.jpg
    ├── recipe.jpg
    └── [your-project-images]
```

## Image Guidelines

- **Project Images**: 1200x630px (16:9 ratio), optimized JPG/PNG
- **Logos**: SVG preferred for scalability
- **Max file size**: 500KB per image
- Use descriptive filenames: `project-name.jpg`

## Free Resources

- Icons: [Lucide Icons](https://lucide.dev) (already included)
- Tech Logos: [Simple Icons](https://simpleicons.org)
- Stock Photos: [Unsplash](https://unsplash.com)
- Illustrations: [unDraw](https://undraw.co)
